# APPENDIX C: TECHNICAL ARCHITECTURE

### Stack
- **Frontend:** Next.js + Tailwind CSS
- **Backend:** Node.js + PostgreSQL
- **AI Engine:** Claude API (Sonnet) + Custom ML models
- **Integrations:** Stripe API, PayPal API, Plaid (bank)
- **Infrastructure:** Vercel + Railway
- **Email/SMS:** SendGrid + Twilio

### Data Flow
1. User connects invoice source (Stripe/PayPal/CSV)
2. AI analyzes invoice + client history
3. ML model predicts payment probability & optimal reminder time
4. AI generates personalized reminder message
5. Message sent via preferred channel
6. Response tracked → model retrains

### Security
- SOC 2 Type II certified (in progress)
- AES-256 encryption at rest
- TLS 1.3 in transit
- GDPR compliant
- No payment credentials stored
